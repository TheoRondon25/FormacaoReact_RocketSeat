# Formação React — Rocketseat 🚀

Este repositório reúne **toda a minha jornada de aprendizado em React pela Rocketseat**, concentrando **todos os projetos desenvolvidos ao longo da formação em um único lugar**.

A ideia aqui é simples: **um repositório único para a formação**, organizado por módulos, facilitando a evolução, revisão de conteúdo e acompanhamento do meu progresso.

---

## 📂 Organização do Repositório

Cada pasta representa um **módulo, aula ou tema específico** abordado durante a formação.

Exemplos de organização:

```
📁 HelloWorld --> primeiros modelos com principios básicos 
```

* Cada pasta é **independente**
* Pode conter um ou mais projetos
* Reflete exatamente o conteúdo estudado naquele módulo

---

## 🎯 Objetivo

Este repositório tem como objetivo:

* Consolidar todo o aprendizado da formação em React
* Servir como **material de estudo e revisão**
* Demonstrar evolução técnica ao longo do tempo
* Manter um histórico real de aprendizado (commits, refactors e melhorias)

👉 **Não serão criados novos repositórios para projetos da formação.**
👉 Projetos em repositórios separados existirão **apenas para projetos práticos, desafios ou exercícios independentes**.

---

## 🛠️ Tecnologias e Conceitos

Ao longo da formação, este repositório pode conter exemplos práticos de:

* React.js
* JavaScript (ES6+)
* TypeScript
* Componentização e reutilização de código
* Hooks (useState, useEffect, etc.)
* Consumo de APIs REST
* Gerenciamento de estado
* Estilização (CSS, Styled Components ou Tailwind)
* Boas práticas de código
* Clean Code no front-end
* Versionamento com Git

---

## 📈 Evolução Contínua

O código aqui **não representa apenas versões finais**, mas sim a evolução natural do aprendizado:

* Melhorias
* Refatorações
* Ajustes de arquitetura
* Correções de conceitos

Isso faz parte do processo 🚀
